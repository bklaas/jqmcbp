There's a lot to learn about Big Red, as noted in the Western Kentucky Sports Marketing Guidelines PDF, which I definitely didn't read in its entirety.

  • Big Red taunts; is never taunted.
  • Big Red cannot eat, but does chew Big Red gum and drinks exclusively Code Red.

  • Big Red's favorite color is ironically blue.
  • ONCE AGAIN BIG RED DOES NOT TALK PLEASE DO NOT PORTRAY TALKING WITH THE USER OF CONVERSATION BALLOONS.
